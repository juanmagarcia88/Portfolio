Aquí voy a mostrar algunos de los proyectos en los que voy a trabajar. Cada uno va a reflejar lo que voy aprendiendo y cómo aplico mis conocimientos en el desarrollo web tanto en Front-end como en Back-end.

Web de Reservas del Hotel Casa Azul

Este proyecto fue desarrollado durante mi curso de 2º DAW. Se trata de una aplicación web para simular reservas de habitaciones en un hotel de mi pueblo, Andújar. Los usuarios pueden registrarse, iniciar sesión, realizar reservas según fechas y tipos de habitaciones, y consultar o eliminar sus reservas desde su área personal. El administrador puede visualizar y gestionar todas las reservas.

  • Front-end: Vue.js
  • Back-end: Spring Boot
  • Base de datos: MySQL

Aprendí muchísimo desarrollándolo: integración entre frontend y backend, gestión de usuarios y autenticación, y comunicación con base de datos. Fue un proyecto grande que me ayudó a afianzar muchos conocimientos.

También implementé responsive y adapté toda la web a cualquier tamaño de pantalla.
